# Auto Scenario Video Builder

This repository contains the complete source code for the Auto Scenario Video Builder project.  
The tool provides a command‑line interface (`autovideo`) that converts text scripts into short vertical or horizontal videos suitable for social media platforms like Reels, TikTok and YouTube Shorts.

Key features include:
- Multiple audio modes (silent, music, voiceover, voiceover+music)
- Multiple text modes (none, subtitle, title cards) and subtitle animation styles
- Configurable presets for different platforms and styles
- A doctor command to check FFmpeg installation and API keys
- Optional integration hooks for AI providers to improve scene planning
- Batch processing, social media caption generation and more

See the `docs/usage.md` file for installation and usage instructions.
