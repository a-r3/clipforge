## Roadmap

This roadmap outlines planned milestones and potential enhancements for `clipforge`.

1. **Basic MVP** – Complete a working CLI with manual scene planning, stock media search and video assembly. (Done)
2. **Batch Processing** – Implement `init-batch` and `batch` commands for multiple jobs. (Done)
3. **AI Assistance** – Integrate optional AI providers for scene planning and social pack generation. (In progress)
4. **GUI Studio** – Develop a graphical editing interface accessible via the `studio` subcommand. (Planned)
5. **Direct Publishing** – Add a module to publish videos and metadata to social platforms. (Planned)
