## Features

The `autovideo` tool provides a comprehensive workflow for turning text scripts into short‑form videos. Key features include:

* Scene splitting and planning via a script parser and optional AI assistance.
* Automatic search for royalty‑free stock media based on generated queries.
* Configurable audio modes: silent, music, voiceover and combinations thereof.
* Multiple text modes for subtitles or title cards, with selectable animation styles.
* Platform presets for Instagram Reels, TikTok, YouTube Shorts and more.
* Batch processing, social media pack generation, and export bundle creation.
